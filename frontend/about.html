About Waste-AI

Waste-AI is a lightweight AI assistant to identify the waste type (organic, recyclable, hazardous, general) from a photo and provide disposal advice. It's perfect for hackathons and prototypes — built with a PyTorch model & Flask backend, and an HTML/CSS/JS frontend.

Why this matters

Correct sorting reduces landfill, improves recycling yields and keeps hazardous waste out of municipal streams. This tool helps everyday users quickly know where to dispose an item.

How it works (high-level)

  1. User uploads a photo from the web UI.
  2. Front-end sends image to Flask /predict endpoint.
  3. Backend runs the PyTorch model and returns the class and advice.
  4. Front-end displays the result and optionally saves to history.
